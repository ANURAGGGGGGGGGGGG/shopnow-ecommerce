## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/shopnow-ecommerce.git
```

2. Install dependencies:
```bash
cd shopnow-ecommerce
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser at:
```
http://localhost:3000
```
